# TA!Dusttale

Source code for [Team Ashes' dusttale](https://gamejolt.com/games/sans_need_to_chill_out_fr/1029475)!

As of 11/28/2025, i wouldn't play this...? there really is no game (yet)

This was built in the latest version of GameMaker BETA.
[gamemaker beta for windows](https://gamemaker.io/en/download/windows/beta/GameMaker.exe)  
[gamemaker beta for mac](https://gamemaker.io/en/download/mac/beta/GameMaker.pkg)  
[gamemaker beta for ubuntu](https://gamemaker.io/en/download/ubuntu/beta/GameMaker.zip)  
Once you've installed GameMaker, download this repo as a zip, extract it, then double-click the TA!Dusttale.yyp file. It should open in GameMaker!!
